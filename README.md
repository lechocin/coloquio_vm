# TP - Máquinas Virtuales

## Descripción
Este repositorio contiene el proyecto práctico para el coloquio de **Máquinas Virtuales**. 
Demuestra el uso de Linux Mint dentro de VirtualBox y el control de versiones con Git y GitHub.

## Estructura
- `comandos-linux.txt`: Lista de comandos esenciales de Linux
- `notas/`: Resumen de conceptos sobre virtualización

## Tecnologías
- VirtualBox (hipervisor)
- Linux Mint 22.3 (SO invitado)
- Git / GitHub (control de versiones y trabajo en equipo)
