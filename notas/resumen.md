# Resumen - Máquinas Virtuales

## ¿Qué es una VM?
Una máquina virtual es un software que emula una computadora completa (CPU, RAM, disco, red) dentro de otra, de forma aislada.

## Herramientas usadas
- **VirtualBox**: Hipervisor tipo 2 (corre sobre Windows)
- **Linux Mint 22.3**: SO invitado (basado en Ubuntu)
- **Windows**: SO anfitrión

## Ventajas
- Probar sistemas sin afectar el equipo real
- Aprendizaje de Linux sin riesgos
- Snapshots para volver a estados anteriores
